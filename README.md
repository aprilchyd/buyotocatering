Fitur Utama dalam Kode:
Visual Mewah: Menggunakan font Playfair Display dan Lato, dengan skema warna gelap dan emas.
Simulasi Video Hero: Efek Ken Burns (slow zoom) pada gambar latar belakang untuk menggantikan video berat, memberikan kesan sinematik.
Copywriting Eksklusif: Tidak ada harga yang ditampilkan secara gamblang, menggunakan istilah "Bespoke Services" dan "Concierge".
Integrasi WhatsApp VIP: Tombol yang memicu pesan formal otomatis sesuai permintaan.
Peta Kustom (Dark Mode): Google Map diset dengan CSS filter agar sesuai dengan tema hitam/emas website.
Tanpa Alert Bawaan: Menggunakan notifikasi Toast dan Modal kustom yang elegan.
Perubahan Utama:
Palet Warna: Menggunakan latar belakang Cream/White yang bersih dan hangat.
Tipografi: Warna teks diubah menjadi gelap (Dark Grey) untuk keterbacaan maksimal di latar terang.
Visual: Bagian Hero tidak lagi gelap/dramatis, melainkan cerah, bersih, dan menenangkan (Airy & Spacious).
Komponen: Kartu dan formulir menggunakan gaya Glassmorphism putih transparan dengan bayangan halus (Soft Shadow).
